# Welcome to Open Chat Docs!

> Docs are still work in progress feel free to open a PR

Generaly the repo comes with 3 services:

- `backend`: <Link href="https://www.djangoproject.com/" target="_blank">Django</Link> + Rest-Framework + Django-Channels
- `frontend`: Express + Vike
- `proxy`: Nginx

All components are containerized, check <Link href="/docs/dev-docker">docker development</Link>.
