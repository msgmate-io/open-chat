# Welcome to Open Chat Docs!

> Documentation is still a work in progress. Feel free to open a PR.

This repository comprises of three services:

- `backend`: Django + Rest-Framework + Django-Channels.
- `frontend`: Express + Vike.
- `proxy`: Nginx.

All components are deployed in containers. Check the docker development guide for more details.
